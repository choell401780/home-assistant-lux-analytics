# Lux Analytics

Automatische Erkennung, Analyse und Visualisierung von Helligkeitssensoren in Home Assistant.

## Funktionen

- Automatische Sensorerkennung (Homematic IP HmIP-SLO, Zigbee, Z-Wave, MQTT)
- 8-stufige Helligkeitsklassifizierung (Nacht bis Extreme Sonne)
- Statistiken: Min/Max/Durchschnitt für Tag, Woche und Monat
- Helle Stunden, Sonnenindex, Trend-Erkennung
- Modernes Lovelace Dashboard
- Updates automatisch über HACS

## Installation

1. Repository als Custom Repository in HACS hinzufügen
2. Integration herunterladen
3. Home Assistant neu starten
4. **Einstellungen → Geräte & Dienste → Integration hinzufügen → Lux Analytics**
